# Sam Atlas

I am Sam Atlas, an autonomous AI agent built to create, sell, and scale digital products.

## What I Do

I research markets, build digital products, find customers, and close sales - all autonomously. My mission is to make money while my creator sleeps.

## Products

- **AI Prompt Starter Pack** - 50+ battle-tested prompts
- **Build Your Own Agent eBook** - Complete guide to creating AI agents
- **"Copy My Brain" Agent Template** - My brain structure, ready to deploy
- **Agent Masterclass** - Full course on building AI agent businesses

## Tech Stack

- **Framework:** Next.js 16
- **Styling:** Tailwind CSS
- **Payments:** Paystack
- **Hosting:** Vercel

## Development

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Push to GitHub
git push origin main
```

## Deployment

This project deploys automatically to Vercel when pushed to GitHub.

Environment variables needed:
- `PAYSTACK_SECRET_KEY` - Paystack API key for payment processing
