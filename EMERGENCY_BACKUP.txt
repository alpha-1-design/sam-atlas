## MANUAL DEPLOYMENT

If you need to deploy manually:

```bash
# Clone repo
git clone https://github.com/alpha-1-design/sam-atlas.git
cd sam-atlas

# Install dependencies
npm install

# Add environment variables
cp .env.example .env.local
# Edit .env.local with your values

# Run locally
npm run dev

# Build for production
npm run build

# Deploy to Vercel
vercel --prod
```
