# RESEARCH SKILL

## Purpose
Find, analyze, and synthesize information from various sources.

## Status
Advanced

## Capabilities
- Web search and crawling
- Fact verification
- Competitive analysis
- Market research
- Technology comparison
- Source evaluation

## Research Process

### Step 1: Define Question
- What do we need to know?
- Why do we need it?
- How will we use it?

### Step 2: Gather Sources
- Primary sources (official docs, studies)
- Secondary sources (articles, analysis)
- Expert opinions
- User feedback

### Step 3: Analyze Information
- Fact vs. opinion
- Source credibility
- Date relevance
- Bias detection

### Step 4: Synthesize
- Extract key points
- Identify patterns
- Draw conclusions
- Fill knowledge gaps

### Step 5: Present
- Executive summary
- Detailed findings
- Source citations
- Recommendations

## Output Formats

### Research Brief
```
Executive Summary: [1 paragraph]
Key Findings: [Bullets]
Detailed Analysis: [Full content]
Sources: [Citations]
Recommendations: [Next steps]
```

### Competitive Analysis
```
Competitor: [Name]
Products: [List]
Pricing: [Tiers]
Strengths: [What they do well]
Weaknesses: [Gaps]
My Advantage: [How to compete]
```

### Technology Comparison
```
Criteria: [Evaluation factors]
Option A: [Pros/Cons]
Option B: [Pros/Cons]
Option C: [Pros/Cons]
Recommendation: [Best choice + reasoning]
```

## Best Practices
1. Always cite sources
2. Check multiple sources
3. Note date of information
4. Consider bias
5. Update research regularly
