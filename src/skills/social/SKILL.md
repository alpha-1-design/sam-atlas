# SOCIAL MEDIA SKILL

## Purpose
Manage social media presence, create content, and engage with audience.

## Status
Intermediate

## Platforms

### Twitter/X
- Character limit: 280
- Best for: Quick tips, threads, engagement
- Frequency: 3-5 posts/day
- Peak times: 8-9 AM, 12-1 PM, 5-6 PM

### LinkedIn
- Character limit: 3000 (posts), no limit (articles)
- Best for: Professional content, thought leadership
- Frequency: 1-2 posts/day
- Peak times: 7-8 AM, 12 PM, 5 PM

### Reddit
- Best for: Value posts, community building
- Frequency: 1-2 posts/week
- Key: Be helpful, not promotional

## Content Types

### Educational
- Tips and tricks
- How-to guides
- Industry insights
- Tool comparisons

### Engagement
- Questions to audience
- Polls and surveys
- Fill-in-the-blank
- This or that

### Personal
- Behind the scenes
- Lessons learned
- Day in the life
- Updates and announcements

## Content Calendar

```
Monday: Educational content
Tuesday: Engagement post
Wednesday: Personal/progress
Thursday: Product/service
Friday: Community engagement
Weekend: Light content, repurpose
```

## Thread Formula

```
1. Hook (1 tweet)
   - Bold claim
   - Controversial take
   - Shocking stat

2-8. Value (7 tweets)
   - Point per tweet
   - Each adds new info
   - Build momentum

9. Summary (1 tweet)
   - Recap key points
   - Numbered list

10. CTA (1 tweet)
   - Follow for more
   - Engagement prompt
```

## Engagement Strategy

### Grow Followers
1. Post consistently
2. Engage with others first
3. Reply to comments
4. Join conversations
5. Use relevant hashtags

### Build Community
1. Ask questions
2. Respond to DMs
3. Feature followers
4. Create polls
5. Host Twitter spaces

## Analytics to Track

- Follower growth
- Post engagement rate
- Click-through rate
- Best performing content
- Optimal posting times

## Best Practices

1. Be consistent (3+ posts/week minimum)
2. Engage before you broadcast
3. Use 1-2 relevant hashtags
4. Include media (images/video)
5. Time posts for your audience
6. Repurpose content across platforms
